All scripts used to train, validate, and deploy SEDflow are publicly available in the github repo: https://github.com/changhoonhahn/SEDflow . It includes:

  • bin/train/anpe.py: script used to train SEDflow. Requires pytorch and the sbi package.

  • bin/nsa.py: script used to deploy SEDflow on optical photometry of NSA galaxies

  • bin/training_data.ipynb: notebook used to compile and construct training data

  • docs/nb/: notebooks used to generate all the results for Hahn & Melchior (2022).

All of the data used to train and test SEDflow is also publicly available at .

  • data.v1.*: training data

  • test.v1.*: test data
